### ⛩️ Installation

Press 🪟 Win + R \
Paste this:

```
wt.exe powershell.exe irm https://raw.githubusercontent.com/MrGrappleMan/Fynelium-NT/main/script/Launcher.ps1 | iex

```
Press `Ctrl` + `Shift` + `Enter`
